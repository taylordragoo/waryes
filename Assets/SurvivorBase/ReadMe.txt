Thank you for the purchase! By default the asset made using build in pipeline
Dependency is Post-Processing and Unity Splines 

To change the SRP please follow the intruction


HDRP

Unzip Main HDRP pack by double click it, press import
Assign HDRP High Fidelity 1 in Project Settings-Quality

Unity 6+ Use HDRP-Unity6 after HDRP pack


URP

Unzip Main URP by double click it, press import

For Unity 6 Use URP-Unity6 after URP pack
For Unity 6.1 Use  Fix_URP_Unity6 and URP_Shaders_6.1 after URP pack
Assign URP Render Asset in Project Settings- Quality

On request I can provide 4k textures suits for film/cinematics

To interact with doors, add DoorHandler script to your Character Controller

If you have Visual Fidelity Pro:
Important! Before importing it, make sure you have all effects from the bundle inside your project
Uzip SB_Visual Fidelity Pro file to have the preset imported (Unity 6+)
Assign PC_RPAsset 1 to Quality tab, this way you will have all effects applied for you

If you have Realistic Car Controller Pro asset you can unzip RCCP_Prefabs to have ready made prefabs of Pickup and Van
