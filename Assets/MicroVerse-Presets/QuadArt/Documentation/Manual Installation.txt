Description
------------------

Some presets require manual installation.

Reason: You'd get prefab guid errors upon installation of Biomes and Presets for MicroVerse if you don't have all the required files installed.

Example: Presets for MicroVerse Roads require Unity Splines. If you have neither of these installed, then the spline for area effects in the prefab would be missing and Unity would show an error in the console during import.

It's either you ignore these errors during import or you import the file manually in case you want to use it. Since errors are confusing upon installation, the decision for manual installation was made.
